# V2 Components

Reusable UI components for the v2 portfolio.

- `V2Layout.jsx` - Main layout wrapper (header, nav, footer)
- `V2Nav.jsx` - Navigation menu
- `ProjectCard.jsx` - Project card with expandable details
- `TimelineItem.jsx` - Timeline entry component
- `ExpandableText.jsx` - Progressive disclosure for long text
- `ArrowIcon.jsx` - Modern arrow icons (down, left, right)
