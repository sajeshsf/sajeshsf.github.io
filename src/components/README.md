# V2 Components

Reusable UI components for the v2 portfolio.

## Core Components

- **V2Layout.jsx** - Main layout wrapper with header, navigation, footer, and error boundary
- **V2Nav.jsx** - Navigation menu with smooth scroll support
- **SocialLinks.jsx** - Social media links (LinkedIn, Email, Phone, GitHub) with modern SVG icons

## Display Components

- **ProjectCard.jsx** - Project card with chevron indicator, links to detail pages
- **TimelineItem.jsx** - Timeline entry component for experience/education
- **TimelineGrid.jsx** - Timeline container with progressive disclosure
- **ProjectsGrid.jsx** - Projects grid with progressive disclosure

## UI Components

- **ExpandableText.jsx** - Progressive disclosure for long text with fade-out
- **ArrowIcon.jsx** - Modern SVG arrow icons (down, left, right)
- **ErrorBoundary.jsx** - React error boundary for graceful error handling

## Props

All components use PropTypes for type safety. See individual component files for prop definitions.
