# Components

Reusable UI components for the portfolio website.

## Overview

This directory contains all reusable UI components. Components are organized by purpose:
- **Layout Components** - Layout.jsx, Nav.jsx, ErrorBoundary.jsx
- **Display Components** - TimelineGrid.jsx, TimelineItem.jsx, ProjectsGrid.jsx, ProjectCard.jsx, BlogCollection.jsx
- **UI Components** - SEO.jsx, SocialLinks.jsx, ExpandableText.jsx, RotatingTagline.jsx, ArrowIcon.jsx

## Principles

- Use PropTypes for type checking
- Include ARIA labels and semantic HTML for accessibility
- Ensure responsive design for all screen sizes
- Follow React best practices (hooks, functional components)
