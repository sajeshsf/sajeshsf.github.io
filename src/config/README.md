# Config

Configuration and constants for the portfolio website.

## Overview

This directory contains centralized configuration in **constants.js**.

## Key Constants

- Progressive disclosure settings (INITIAL_PROJECTS_COUNT, INITIAL_TIMELINE_ITEMS, etc.)
- Scroll animation settings (SCROLL_PARALLAX_FACTOR, HERO_SCROLL_THRESHOLD, etc.)
- Page IDs (PAGE_IDS object)
- Intersection Observer settings

## Usage

Import constants where needed:
```javascript
import { INITIAL_PROJECTS_COUNT, PAGE_IDS } from './config/constants.js'
```

For complete configuration documentation including all constants and their values, see [src/README.md](../README.md#config).
