# Utils

Utility functions and React hooks for the portfolio website.

## Overview

This directory contains:
- **Hooks**: useHash.js, useScroll.js, useProgressiveDisclosure.js
- **Utilities**: slugify.js

## Quick Reference

- `useHash()` - React hook for hash-based routing
- `useScroll()` - Tracks window scroll position for parallax effects
- `useProgressiveDisclosure()` - Hook for expand/collapse patterns with fade-out truncation
- `slugify()` - Converts strings to URL-friendly slugs
