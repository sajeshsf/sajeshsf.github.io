# V2 HTML Entry Points

HTML entry files for Vite multi-page build.

Each subdirectory contains an `index.html` that loads the corresponding React page component via `src/v2/main.jsx`.

**Routes:**
- `/v2/` - Home page
- `/v2/about/` - About page
- `/v2/experience/` - Experience page
- `/v2/projects/` - Projects page
- `/v2/writing/` - Writing page
